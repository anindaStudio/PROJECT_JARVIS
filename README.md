# 🤖 JARVIS - AI Voice Assistant

JARVIS is a Python-based AI voice assistant inspired by Iron Man's virtual assistant. It can perform tasks like voice interaction, opening applications, searching the web, and executing commands.

---

## 🚀 Features

* 🎤 Voice recognition (Speech to Text)
* 🔊 Text to speech response
* 🌐 Web search (Google, YouTube, etc.)
* 📂 Open applications (Chrome, VS Code, etc.)
* ⏰ Tell time and date
* 📧 Basic automation tasks
* 🧠 Custom command support

---

## 🛠️ Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* pyaudio
* webbrowser
* datetime
* os

---

## 📦 Installation

```bash
git clone https://github.com/anindaStudio/PROJECT_JARVIS.git
cd PROJECT_JARVIS
pip install -r requirements.txt
```

---

## ▶️ Usage

```bash
python main.py
```

Then say: **"Hey Jarvis"** to activate.
